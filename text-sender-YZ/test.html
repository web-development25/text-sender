Learning HTML:

Structure:

The structure is one of the most important things in a html, without it the whole code does not work correctly. If you are on visual studio code, type html:5 or doc which in this case they're emmet abreviations. They automatically type the code for you.

The structure used in HTML is: 
            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Document</title>
            </head>
            <body>
            
            </body>
            </html>
This is one of the options of structure you need, once you learn the syntaxis (the way the code is written), you wont need to take a look at the structure again.

Classes in HTML:

            <div class="Name Of Class"> text here </div>
This is the correct way to make classes. Classes are a custom group in which you can put any string characters, to call it later on CSS. You can re-use the classes by doing the same. Some examples are:

<div class="TextNo1"> <h>Hello World</h>
<div class="TextNo1"> <h1>This is a header.</h1>
<div class="TextNo1"> <h2>This is a subtitle.</h2>
